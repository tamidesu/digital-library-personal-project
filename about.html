What this app demonstrates

  • IndexedDB data layer for products and users
  • Auth UI with persistent session via localStorage
  • Cart → Checkout with validation and confirmation
  • Purchase history tied to the current user
  • Reusable layout via includes (header/footer)

Privacy & Storage

This is a lab/demo project. Data is stored locally in your browser (IndexedDB + localStorage). In production systems, passwords must be hashed and sensitive data must never be stored in plain text.
